Poirot is a data allocation/filtering tool with a user-friendly, yet powerful interface

Poirot is a tool that lets you import table data (any sort of csv, or other gridlike data should do). It then allows you to create and compose conditions, in order to split up your data any way you like. Finally after a suitable split-up, you can also perform aggregations/totals/etc for each split performed by your tree of formulas.
